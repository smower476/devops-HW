# C++ REST Server with PostgreSQL

Этот проект представляет собой высокопроизводительный REST-сервер на языке C++, работающий с базой данных PostgreSQL. Проект полностью контейнеризирован и готов к развертыванию в рамках DevOps-практик.


## Структура проекта
* `main.cpp` — исходный код сервера (эндпоинты и логика БД).
* `CMakeLists.txt` — конфигурация сборки проекта.
* `Dockerfile` — multi stage сборка.
* `docker-compose.yml` — оркестрация контейнеров приложения и базы данных.
* `.env` — файл с переменными окружения (настройки подключения).
* `init.sql` — скрипт для автоматического создания таблиц при первом запуске.
